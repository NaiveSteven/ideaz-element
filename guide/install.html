使用包管理器 ​

我们建议您使用包管理器（如NPM、Yarn 或 pnpm）安装 Ideaz Element，然后您就可以使用打包工具，例如 Vite 或 webpack。

sh
npm install ideaz-element --save
sh
yarn add ideaz-element
sh
pnpm install ideaz-element
